## Getting Started

Setup environment

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

NOTE: Don't forget to start the backend server as well!

## Frontend Stack
